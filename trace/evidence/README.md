# TRACE Evidence

This directory contains curated Light Curves and handoffs for adopting projects.

- Name Light Curves `<TASK-ID>_LIGHT_CURVE.md`.
- Name handoffs `<TASK-ID>_HANDOFF.md` and `<TASK-ID>_HANDOFF.json`.
- Commit concise results, traceability, selected screenshots, and review verdicts.
- Store raw logs, traces, videos, coverage HTML, generated packages, and bulk screenshots as Actions artifacts.
- Never commit secrets, cookies, browser profiles, private keys, personal data, or unredacted sensitive output.
